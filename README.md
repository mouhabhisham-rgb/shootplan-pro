# ShootPlan Pro — الإنتاجات البسيطة

أداة تخطيط التصوير الاحترافية لـ Simple Productions.

## النشر
منشور على Vercel — ملف HTML واحد لا يحتاج build أو dependencies.

## الميزات
- استيراد ذكي بالذكاء الاصطناعي
- كول شيت + تصدير PDF
- مشاركة عبر رابط أو واتساب
- حفظ في السحابة (Supabase)
- عربي RTL كامل
- يعمل على الجوال والديسكتوب

## المتطلبات
- Supabase project (مربوط بالفعل)
- Vercel account

## الإعداد
1. ارفع الملفات على GitHub
2. اربط الـ repo بـ Vercel
3. أضف رابط Vercel في Supabase → Authentication → URL Configuration
